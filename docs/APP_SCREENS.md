# 월급마을 화면 명세

> 시즌1. 이번달 생존기 — 정적 UI (mock data)

## 네비게이션 구조

```
[마을 홈] ←→ [내 방] ←→ [광장] ←→ [방명록]
     ↓
[생존카드] / [오늘의 퀘스트]  (홈에서 진입)
```

### 하단 탭 (BottomNav)

| 탭 | 경로 | 아이콘 |
|----|------|--------|
| 마을 | `/` | 🏘️ |
| 내 방 | `/room` | 🏠 |
| 광장 | `/square` | 📢 |
| 방명록 | `/guestbook` | 📝 |

---

## 1. 마을 홈 (`/`)

**목적:** 앱 진입점. 오늘의 생존 현황 한눈에 + 주요 메뉴 타일

### 구성

```
┌─────────────────────────┐
│  PixelHeader            │
│  "월급마을" + 시즌 뱃지   │
├─────────────────────────┤
│  CharacterAvatar (큰)   │
│  + SpeechBubble         │
│  "오늘의 한마디"          │
├─────────────────────────┤
│  D-day 카운터           │
│  "월급날까지 D-12"       │
├─────────────────────────┤
│  SurvivalBadge + 상태   │
│  생존등급 B / 오늘: 빡빡  │
├─────────────────────────┤
│  위기 카테고리 뱃지       │
│  ☕ 커피  🍜 배달        │
├─────────────────────────┤
│  [생존카드] [오늘의퀘스트]│
│  2열 타일 메뉴           │
├─────────────────────────┤
│  오늘의 마을 소식 (mock)  │
└─────────────────────────┘
│  BottomNav              │
└─────────────────────────┘
```

### 공개 정보만 표시

- D-day, 생존등급, 오늘 상태, 위기 카테고리, 한마디, 캐릭터 HP

## 6. 오늘의 퀘스트 (`/quest`)

**목적:** 월급 생존을 게임 퀘스트 메타포로 — 오늘 할 일 목록

### 구성

```
┌─────────────────────────┐
│  PixelHeader "오늘의 퀘스트"│
│  D-12 · 등급 B          │
├─────────────────────────┤
│  QuestItem × N          │
│  ┌───────────────────┐  │
│  │ ☐ 점심 도시락 챙기기│  │
```

- 유머러스한 퀘스트 문구
